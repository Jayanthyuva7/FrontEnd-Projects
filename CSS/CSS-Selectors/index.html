Selector Feature Comparison

Selector Purpose Syntax Example
Class Selector Styles elements with a specific class .item
ID Selector Styles a unique element with an ID #main
Element Selector Styles all elements of a specific type ul, li, p
Attribute Selector Styles elements with a specific attribute input[type="text"]
